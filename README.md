# Science Bulletin Reproducibility Package (RIEC)

This repository hosts the reproducibility materials for our Science Bulletin submission. (Title
Auditable Plant-Level Digitalization Diagnosis under Clustered Observations: Measurement Audit, Virtual-Factory Benchmarking, and Guarded RIEC
)


## Quick start
1) Download `reproducibility guide.zip`
2) Unzip locally
3) Follow the runbook inside the bundle

## Environment
- OS: macOS 14.x (Sonoma)
- Python: 3.13
- Reproducibility notes and dependencies are documented inside the bundle.
